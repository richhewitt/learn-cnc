Tool Changers

For simpler CNC machines, each tool can be manually installed into the spindle when it is needed. High production machines, however, require fast turnover between different tools. Different magazines have been developed to hold a multitude of tools for quick and automatic changeover.

Stationary Tool Bank

This is likely the simplest tool changer for CNC machines. Tools are held in a row on one side of the machine. First, the main spindle head travels to the tool needed.  It then lowers and connects onto the tool holder.  Once the tool holder is secured, the spindle moves away taking the tool with it.  These steps are reversed to bring the tool back to the magazine. 

Chain Tool Changer

In a chain-type tool changer, tools are moved into position using a traveling chain.  Once in position, the spindle can connect to the tool holder.  This system has the potential to hold a large number of tools.

Rotary Tool Changer

A rotary toolchanger in motion

A rotary toolchanger holds tools in a circular fashion, rotating the correct tool to the electrospindle.

Traveling Tool Changer

This type of tool changer travels with the spindle as it is machining.  Sometimes only a few tools are needed to machine a simple part, but many tools are needed to continuously operate the machine.  In these cases, several tools can be installed in a traveling tool changer while other, less frequently used tools are held in a chain or rotary tool changer.
