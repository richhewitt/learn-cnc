The LearnCNC Project

Free educational content on CNC machines and tooling. Topics include:

  • Electrical Components of CNC
  • Mechanical Components of CNC
  • Cutting Applications of CNC routers
  • Tooling Physics
  • Tool Design
  • Tool Metallurgy

    • Developed in 2009 with generous funding from BC Campus and developed by:

  • Richard Hewitt - Project Manager
  • Robert Shum - Software Engineer
  • Nikita Donets - Software Engineer
  • Jason Beatch - 3D Modeler and Animator

    • There is also an interactive 3D environment to explore this content.
    It is a very old MSI package which may not work on modern Windows versions.
